# Archive — historical web work

Everything in this folder documents how the site arrived at the locked 2026-04-20 source of truth. Nothing here is live. Do not edit these files; they are frozen for reference only.

## Folders

### `2026-04-19_build/`
First build rounds. Each `build_log_*.md` captures what was attempted that round, each `claude_code_round*_fix_prompt.md` is the prompt used to brief the next iteration, and each `staging_qa_*.md` is a QA pass. `verification_report.md` and `claude_code_build_brief.md` are the original kickoff docs.

### `2026-04-20_audit/`
The morning audit that flagged the first build as too dark / too small / too bland, the SEO audit that ran alongside, the SEO fix log, and the round 8 redesign prompt that triggered the crest-forward rebuild.

### `superseded_drafts/`
- `design_direction.md` — original design direction doc. Superseded by the approved direction captured in project memory (`design_direction_approved.md`) and by `SOURCE_OF_TRUTH.md`.
- `webflow_head_paste.html`, `webflow_head_paste_split.html` — earlier versions of the custom-code paste. The current canonical version is `../webflow_head_paste.html` (formerly `webflow_head_paste_round8_clean.html`).
